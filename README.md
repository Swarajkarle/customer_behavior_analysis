# customer_behavior_analysis

📊 Data Analytics Project
📌 Overview

This project demonstrates an end-to-end Data Analytics workflow, starting from raw dataset loading to building interactive dashboards and presenting business insights.

The project focuses on:

Data loading and preprocessing using Python

Performing Exploratory Data Analysis (EDA)

Data cleaning and transformation

Running SQL queries in relational databases

Building an interactive Power BI dashboard

Generating a detailed analytical report

Creating a professional presentation using Gamma

This project highlights practical skills in data analysis, SQL, visualization, and business intelligence reporting.

The dataset contains structured data relevant to business operations.
